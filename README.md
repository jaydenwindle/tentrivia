## Running Locally

1) Clone the repo and `cd` into it:
  ```bash
  $ git clone https://github.com/jaydenwindle/tentrivia.git && cd tentrivia
  ```
2) Install dependencies:
  ```bash
  $ yarn
  ```
3) Run the app on your preferred platform:
  ```bash
  $ yarn <ios | android | web>
  ```
